В репозитории лежит файл по которому открывается локальный сайт "ПрофНавигатор".
Для того чтобы открыть сайт:
1. Скачайте архив
2. Перетащите папку внутри архива на рабочий стол
3. С помощью правой мыши по "index3.html" открыть в удобном вам браузере
4. Готово!
5. В readme по сути написано тоже самое что и тут.
